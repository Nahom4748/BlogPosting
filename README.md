# BlogPosting

# Blog Platform Backend

## Tech Stack

- Node.js
- Express.js
- MySQL/Sequelize
- JWT Authentication
- Jest/Supertest

## Setup

1. Clone repository
2. Install dependencies: `npm install`
3. Create MySQL database
4. Copy `.env.example` to `.env` and configure
5. Start server: `npm start`

## API Endpoints

- POST /api/auth/register
- POST /api/auth/login
- GET/POST /api/blogs
- GET/PUT/DELETE /api/blogs/:id
